# Portafolio Profesional | Duglas Brenes Solis

Sitio web personal de **Duglas Brenes Solis**, Senior Full Stack Developer con experiencia en banca, seguros y finanzas. El portafolio esta disenado como una landing page profesional, minimalista y responsiva, enfocada en conversion para reclutadores y contactos del sector IT.

## Vista General

El sitio presenta:

- Hero profesional con propuesta de valor y CTA para descargar el CV.
- Timeline de experiencia profesional.
- Boveda de habilidades tecnicas por categoria.
- Logros clave relacionados con despliegues automaticos, seguridad 2FA e integraciones Azure.
- Formulario de contacto con correo y telefono.
- FavIcon de marca basado en el monograma `DB`.

## Tecnologias

- HTML5
- Tailwind CSS por CDN
- JavaScript vanilla
- SVG para identidad visual y favicon

## Uso Local

Este proyecto es un sitio estatico. Puedes abrir `index.html` directamente en el navegador.

Tambien puedes servirlo con cualquier servidor estatico local. Por ejemplo:

```bash
npx serve .
```

## Despliegue

El sitio puede desplegarse en cualquier hosting estatico:

- GitHub Pages
- Netlify
- Vercel
- Azure Static Web Apps

Para GitHub Pages, configura la rama `main` como fuente de publicacion y usa la raiz del repositorio como carpeta del sitio.

## Identidad Visual

La marca utiliza una paleta profesional basada en:

- Azul marino profundo: confiabilidad, estabilidad y arquitectura robusta.
- Verde esmeralda `#2D9886`: equilibrio, seguridad y modernidad tecnologica.

El favicon final esta en `favicon.svg`. El archivo `favicon-concepts.svg` contiene cuatro variaciones exploratorias de micro-iconografia.
